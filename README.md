cordova-dnssd
-------------

Cordova plugin for running DNS-SD browsing and resolving on iOS devices.

Instructions
------------

Add *.h, *.m to Plugins/ directory and *.js to www/ directory.  

Add key to Cordova.plist:
 * fi.peekpoke.cordova.dnssd = CDVBonjour
